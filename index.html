Tools

A collection of small web tools, built mostly with the help of LLMs.

github.com/flokoe/tools

  • Bundestag Fortschritt 🇩🇪 – Track the progress of the current German legislative period
  • DMARC Report Analyzer – Analyze DMARC aggregate reports from XML, ZIP, GZ, or TAR.GZ files
  • Gist HTML Renderer – Render raw HTML from a GitHub Gist ID passed in the URL hash
  • Running Interval Timer – Track warmup, run/walk intervals, repetitions, and cues for a running plan
